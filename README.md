# Adidas Sales Analysis Dashboard (Power BI)

##  Project Overview
This project showcases an interactive Power BI dashboard designed to analyze Adidas sales performance. The dashboard provides a comprehensive view of sales, profit, product performance, and regional distribution, enabling data-driven decision-making.
## Objective
* Analyze overall sales and profit performance
* Identify top-performing regions and products
* Understand monthly sales trends
* Evaluate pricing and profit margins
  
## Dashboard Insights

### Key Metrics
* **Total Sales:** $899.9M
* **Total Profit:** $332.1M
* **Units Sold:** 2.5M
* **Average Price per Unit:** $45.2
* **Average Margin:** 42.3%

### Sales Trend Analysis
* Visualized monthly sales using a line chart
* Identified peak sales in mid-year (July–August)
* Observed decline during October followed by recovery

### Regional Performance

* West region contributes the highest sales
* Northeast and Southeast show moderate performance
* Midwest has the lowest contribution

### Product Analysis

* **Men’s Street Footwear** is the top-selling category
* Women’s Apparel also shows strong performance
* Product-wise comparison helps identify demand trends

### Interactive Features

* Region filter for customized analysis
* Date filter for time-based insights
* Dynamic visuals for better user experience

## Tools & Technologies

* **Power BI** – Dashboard development
* **Excel** – Data preprocessing
* **DAX** – Calculations and KPIs
